    class Solution {
	public int longestValidParentheses(String s) {
		int max = 0;
		Stack <Integer> st = new Stack<>();
		st.push(-1);
		for(int i = 0; i < s.length(); i++)
		{
			if(s.charAt(i) == '(')
				st.push(i);
			else
			{
				st.pop();
				if(st.empty() == true)
					st.push(i);
				else
					max = Math.max(max,i - st.peek());
			}
		}
		return max;
	}
}

解題說明

一開始demo以為是很簡單的題目，

直到自己輸入一些複雜的測資才逐漸明白題目的要求，

很直覺的想用數學來解，寫完發現就是會差一點點，

沒想到是要從後面再讀過來一次取max，

用stack比較好呈現這種互相對應的題目，不過我寫得有點水，

而且我沒想到如果只有')'的時候要先POP再PUSH，導致我的stack裡面雜質越來越多。

最後差臨門一腳參考solution才發現可以如此簡潔。
