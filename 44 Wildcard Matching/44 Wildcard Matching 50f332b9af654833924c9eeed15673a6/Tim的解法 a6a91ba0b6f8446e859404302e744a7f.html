    bool isMatch(char * s, char * p){
	char *ss = s,*sp = NULL;
	while(*s) {
		if(*s == *p || *p == '?') {
			s++;
			p++;
			continue;
		}
		if(*p == '*') {
			sp = ++p;
			ss = s;
			continue;
		}
		if(sp) {
			s = ++ss;
			p = sp;
			continue;
		}
		return false;
	}
	while(*p=='*') p++;
	return !*p;
}

解題說明

參考jaby的演算法(還有leetcode上面的)，基本上概念是一樣的

所以我嘗試用不一樣的方法解釋看看

演算法進行中s,p會嘗試找到最長的配對的字串

遇到＊則先假設它對應到空字串

如果對不上，在沒有遇過＊時代表兩字串不配對

若有遇過*則回到上個＊的位置，並讓＊多吃一個字元試試

至於為什麼不用考慮更後面的＊是因為這樣子的演算法會先找到對應相同s長度的p中最短的，所以去前面多吃只會找到更長的

最後當s整個跑完之後,p後面如果是＊就都跳過

如果p也能整個跑完就return true反之則false

ps.'\0'在c可以當作false來判定

⏳ 時間複雜度：O(n)之類的

💾 空間複雜度：O(1)兩個指標
